# Re-entering Transactions

## When to Use
When the main POS system is back online and you need to enter the offline sales.

## Steps

1. **Open Pending Tab** (second tab with badge count)

2. **Select Transaction**:
   - Tap "Re-enter Items" on any pending transaction

3. **Scan Items into Main POS**:
   - Large barcode displays on screen
   - Product name and quantity shown
   - Point phone screen at main POS scanner
   - Main POS reads the barcode

4. **Mark Item Done**:
   - Tap "Done - Next Item"
   - Automatically advances to next item

5. **Complete Transaction**:
   - After last item, confirm "Mark as synced?"
   - Transaction moves to History tab

## Features
- Progress saves automatically (can close app)
- Previous/Pause buttons for navigation
- Barcodes are 300x100dp minimum for easy scanning
- High contrast black on white

## Troubleshooting

### "Barcode won't scan"
- Increase screen brightness
- Clean device screen
- Hold phone steady at 6-12 inches from scanner
