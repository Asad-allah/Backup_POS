# Processing a Sale

## Steps

1. **Open New Sale Tab** (first tab)

2. **Add Items to Cart**:
   - Tap "Scan Barcode" to use camera scanner
   - OR use Search bar to find products by name/barcode
   - Tap product to add to cart

3. **Adjust Quantities**:
   - Use [-] and [+] buttons on each cart item
   - Total updates automatically

4. **Generate Receipt**:
   - Tap "Generate Receipt" button
   - Confirm the sale
   - PDF receipt is saved
   - Transaction saved as "pending"

5. **Clear Cart** (if needed):
   - Tap "Clear" to remove all items

## Notes
- Scanning same barcode twice increments quantity
- Items not found show error with search fallback
- PDF saves to app documents folder
